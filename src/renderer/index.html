Remote harness (over SSH)

Your agent harness runs on another machine (a dev box, a server)? ARIA tunnels to it over SSH so the voice pipeline stays local. For Hermes Agent, use the quick setup, paste the SSH target, then Connect.

Welcome to ARIA

A local-first voice assistant. Speech recognition and speech run on your machine; only the language model is remote. Let's get you set up — takes about a minute.

Choose your agent harness

ARIA routes your voice to an agent harness over an OpenAI-compatible endpoint. Pick one and confirm its endpoint URL.

API key (optional)

If your harness needs a key, paste it here — it's stored encrypted in your OS keyring. Local harnesses usually don't need one; leave it blank.

Add a language model (recommended)

Connect a conversational LLM directly with your own API key — or a local server like Ollama or LM Studio. This answers everyday questions; the agent harness handles tool-using tasks. Set one or both; leave blank to skip.

Microphone access

ARIA listens for a wake word and your voice. Grant microphone access so it can hear you. You can always type instead.

You're all set

Say "hey jarvis" or press the mic button to talk. Change anything later in Settings (gear icon).
